# Folder Structure

## Root
- `app/` — Application entry points, page routes, global layout.
- `components/` — Reusable UI components.
- `config/` — Static application configuration.
- `constants/` — Global constant values used across the application.
- `providers/` — App-level providers and context scaffolding.
- `lib/` — Utility modules, error handling, and validation.
- `types/` — Strongly typed interfaces and shared type definitions.
- `hooks/` — Reusable React hooks.
- `docs/` — Project documentation.

## Recommended future modules
- `features/` — Business domain modules for quote requests, tracking, inventory.
- `services/` — API clients and integration adapters.
- `api/` — Server-side endpoints and request handling.
- `styles/` — Global and shared styling utilities.
